地中の情報を、地上の安心に。

Gio (Ground investigation open data) Vision

広島県が提供するオープンデータプラットフォーム「DoboX」のボーリングデータを活用し、専門知識の有無に関わらず、誰もが直感的に地盤情報を把握できる環境を目指した Web アプリケーションです。

💡 コンセプト

広島県内には膨大なボーリング調査データが存在しますが、それらは専門的な図面（柱状図）やリスト形式で提供されることが多く、一般利用者や非専門の技術者にとっては「アクセス」と「理解」の障壁が高い状態にありました。

GioVision は、以下のミッションを掲げて開発されました。

アクセスの解消: 地図上で今立っている場所のデータを即座に特定する。

理解の解消: 複雑な柱状図を色と形で表現し、直感的に地層を把握する。

✨ 主な機能

1. マップベースの地点探索

Leaflet を採用し、地図を移動させるだけで周囲のボーリング調査地点を動的に表示。目的の地点へ「一発アクセス」が可能です。

2. 簡易柱状図

柱状図を視覚的にわかりやすいバースケールで再構成しました。

色分け: 砂質、粘性土、岩盤などの地層種別を色で識別。

N値の可視化: 硬さを示す N 値をバースケールに重ねて表示

3. 深度目盛りの統合

「深さ」を身体感覚で捉えられるよう、バースケールに整数目盛りを統合しました。

4. 完全レスポンシブ対応

現場での利用を想定し、スマートフォンから PC までデバイスを問わず動作します。

🛠 技術スタック

Frontend: HTML5, Tailwind CSS, JavaScript (ES6+)

Maps: Leaflet.js

Data Source: 広島県 DoboX ボーリングデータ（CSV/JSON形式）

🚀 今後の展望

標高データの連携: 海抜による地層の高さ比較機能。

3D可視化: 地層の連続性を立体的に表現する断面図生成機能。

ハザードマップ統合: 土砂災害警戒区域などとの重ね合わせ表示。
